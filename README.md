# 🌸 Kirby Crush

**Kirby Crush** es un juego desarrollado en **Java (100% vanilla, sin frameworks externos)** como proyecto académico para la materia **Tecnología de la Programación**.

Se trata de un juego estilo match-3 donde el objetivo es completar niveles eliminando combinaciones de fichas de colores, cumpliendo distintos objetivos antes de quedarte sin movimientos.

---

## 🕹️ Mecánicas del juego

- **Niveles:** 5 niveles con dificultad progresiva.
- **Movimientos:** Tenés una cantidad limitada por nivel.
- **Objetivo:** Cada nivel requiere eliminar una cantidad de colores específicos.
- **Victoria:** Completás el nivel si cumplís todos los objetivos antes de quedarte sin movimientos.
- **Derrota:** Perdés una vida si se te acaban los movimientos sin cumplir los objetivos.
- **Vidas:** Comenzás con 3 vidas. Si las perdés todas → **Game Over**.
- **Ranking:** Se guarda el puntaje de los mejores jugadores.

### 🍬 Tipos de entidades
- **Kirby verde.** 
- **Kirby amarillo.**

- **Kirby violeta:**
- **Kirby rosa**
- **Rayado horizontal:** Se obtiene al hacer un match 4 de manera horizontal.
- **Rayado vertical:** Se obtiene al hacer un match 4 de manera vertical.
- **Envuelto:** Se obtiene al hacer un match en forma de L, T, +.
